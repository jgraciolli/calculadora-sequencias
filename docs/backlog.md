# 📌 Backlog do Projeto

## PB-01 - Como usuário do sistema quero visualizar um menu com as opções disponíveis antes de realizar qualquer operação, para poder escolher qual ação desejo executar no sistema
- **Épico:** Interface  
- **Descrição:** O sistema deve apresentar um menu inicial ao usuário antes da execução de qualquer funcionalidade. Esse menu deve listar de forma clara as operações disponíveis, permitindo que o usuário selecione qual ação deseja realizar.  
- **Critério de Aceitação:** Ao iniciar o sistema, deve ser exibido um menu com as operações disponíveis, permitindo que o usuário selecione uma opção válida para executar a funcionalidade correspondente.  
- **Prioridade:** Alta  
- **Status:** Backlog
